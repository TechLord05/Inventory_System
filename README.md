# Inventory Management System

A simple inventory management system built with **Django**, **Django REST Framework**, and **PostgreSQL**.  
Supports authentication, supplier-product management, orders, and stock tracking.

---

## **Features**

- User registration and JWT authentication
- Supplier management
- Product management (associate products with suppliers)
- Orders (incoming/outgoing) with automatic stock updates
- Stock movement history

---

## **Tech Stack**

- Python 3.10
- Django 5.x
- Django REST Framework
- PostgreSQL
- Gunicorn + WhiteNoise (for deployment)
